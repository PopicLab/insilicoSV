# SV_simulator
A simulator for complex structural variants

![Alt text](imgs/complex_sv_classes_diagram.webp)

For the configuration file, use this key when labelling the structural variant:

1 = Insertion\
2 = Deletion\
3 = Inversion\
4 = Duplication\
5 = Translocation\
6 = dupINVdup\
7 = delINVdel\
8 = delINVdup\
9 = dupINVdel\
10 = delINV\
11 = INVdel\
12 = dDUP-iDEL\
13 = INS-iDEL\
14 = dupINV\
15 = INVdup\
16 = dDUP
